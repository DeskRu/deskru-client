# DeskRu Client

Клиент удалённого доступа для инфраструктуры DeskRu.

> Форк [rustdesk/rustdesk](https://github.com/rustdesk/rustdesk) (AGPL-3.0),
> настроенный для работы с серверами DeskRu.

## Скачать

Готовые сборки для Windows, macOS, Linux и Android доступны в
[DeskRu/releases-clients](https://github.com/DeskRu/releases-clients/releases).

## Начало работы

1. Скачайте клиент для вашей платформы по ссылке выше
2. Зарегистрируйтесь на [deskru.ru](https://deskru.ru)
3. Войдите в аккаунт и подключайтесь к устройствам
